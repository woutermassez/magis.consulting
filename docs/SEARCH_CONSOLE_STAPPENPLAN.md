# Google Search Console stappenplan

1. Verwijder foutieve sitemap-inzendingen: `/robots.txt`, `/llms.txt`, `/humans.txt`, `/security.txt`.
2. Dien alleen `sitemap.xml` in.
3. Controleer na Netlify deploy: `https://www.magis.consulting/sitemap.xml`.
4. Gebruik URL-inspectie voor de homepage en vraag indexering aan.
5. Monitor wekelijks: Pagina-indexering, Core Web Vitals, Zoekopdrachten, CTR en Gemiddelde positie.
